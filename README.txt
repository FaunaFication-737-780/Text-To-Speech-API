This is the Text to Speech using google-TEXT-TO-SPEECH Cloud Based API

Step 1: install the node_modules
node install

Step 2: Need to Create API and Key Credential in GOOGLE DEVELOPER CONSOLE, You can create User Authentication Key or Service Account Key

Step 3: Download and Store the key credential where we gonna use to access the API

Step4: Enter the Text in given text.txt file where it need to be converted into Speech

Step 4: Start the Application using
node start

Step 5: The Audio file is saved in your local repository with the name Audio_file in mp3 format.

Enjoy!
